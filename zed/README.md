# Riviera theme for Zed

A delightful dark pastel color theme for the [Zed editor](https://zed.dev).

## Installation

### Manual installation

1. Copy `riviera.json` to your Zed themes directory:
   - **macOS/Linux**: `~/.config/zed/themes/`
   - **Windows**: `%USERPROFILE%\AppData\Roaming\Zed\themes\`

2. Open Zed settings (`Cmd+,` on macOS or `Ctrl+,` on Linux/Windows)

3. Set the theme:
   ```json
   {
     "theme": "Riviera"
   }
   ```

Alternatively, use the theme picker (`Cmd+K Cmd+T` on macOS or `Ctrl+K Ctrl+T` on Linux/Windows) and select "Riviera".
