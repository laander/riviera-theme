# Riviera theme for VS Code / Cursor

A delightful dark pastel color theme for [VS Code](https://code.visualstudio.com) and [Cursor](https://cursor.com).

## Installation

### From the Marketplace

1. Open VS Code or Cursor
2. Go to **Extensions** (`Cmd+Shift+X` on macOS or `Ctrl+Shift+X` on Linux/Windows)
3. Search for **Riviera theme**
4. Click **Install**
5. Open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) and select **Preferences: Color Theme** → **Riviera theme**

### Manual installation

1. Copy the `riviera` folder to your extensions directory:
   - **macOS**: `~/.vscode/extensions/`
   - **Linux**: `~/.vscode/extensions/`
   - **Windows**: `%USERPROFILE%\.vscode\extensions\`

   For Cursor, use `~/.cursor/extensions/` instead.

2. Restart the editor
3. Open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) and select **Preferences: Color Theme** → **Riviera theme**

## Development

The theme is generated from source files in `src/`. To rebuild after making changes:

```sh
npm run build
```
