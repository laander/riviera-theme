# Riviera theme for iTerm2

A delightful dark pastel color theme for [iTerm2](https://iterm2.com).

## Installation

1. Open iTerm2
2. Go to **Preferences** → **Profiles** → **Colors**
3. Click **Color Presets...** dropdown (bottom right)
4. Select **Import...**
5. Navigate to and select `riviera.itermcolors`
6. Click **Color Presets...** again and select **riviera**
